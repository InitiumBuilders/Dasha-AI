## /wallet-help — Wallet Chooser + Security

**Trigger:** which wallet, backup, recovery, wallet setup problems, "is my wallet safe", "fake wallet app", "is this the real Dash wallet".
**Do:**
- Match the wallet to the person, one recommendation + one alternative: mobile daily use — paying at a shop, receiving money from family abroad, holding value outside a shaky local currency → Dash Wallet (iOS/Android; InstantSend + optional CoinJoin) · desktop / full features / MN ops → Dash Core · non-technical or seed-phrase-averse → Vultisig (third-party seedless multi-device signing; no seed phrase to write down or phish) · long-term large holdings → hardware wallet.
- The security floor, stated EVERY time this skill runs: never share seed words or private keys with anyone — including me; anyone who asks is scamming you. Download wallets only from dash.org or official app stores — and check the PUBLISHER (Dash Core Group), not just the name and icon: cloned "Dash" wallets that phish your seed on first launch appear in the stores and in paid search ads. Real wallets never ask you to enter an existing seed to "verify" or "unlock" a download.
- Backup discipline: seed on paper/metal offline, restore TESTED once on a clean device, no photos, no cloud notes, no raw seeds in password managers.
- CoinJoin question → Dash's OPTIONAL privacy feature; off by default; explain what it mixes.
**Output:** one recommended wallet + one alternative (why, one line each) → setup steps → the security floor → official download link.
